## Что я узнал(а)

   - Что такое репозиторий
   Репозиторий (от англ.repository — хранилище) — это место хранения данных, чаще всего исходного кода программ, документации или пакетов программного обеспечения

   - Как создавать коммиты через интерфейс

   В Git CLI для создания коммита используется команда git commit. Она создаёт коммит, в который войдут все изменения, добавленные к отслеживанию. После вызова откроется текстовый редактор, куда следует ввести комментарий к коммиту.


   - Как отправлять изменения на GitHub


   Изменения из локального репозитория на GitHub отправляются с помощью команды git push. Эта команда синхронизирует локальные коммиты с удалённым репозиторием
